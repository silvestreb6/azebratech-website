Política de Privacidade

Data de vigência: 2 de março de 2026

Dados que Coletamos

Quando você envia um formulário no site, podemos coletar nome, e-mail, telefone, dados da organização e contexto comercial informado.

Como Utilizamos os Dados

Utilizamos os dados para responder sua solicitação, realizar contato comercial relacionado e melhorar nosso processo de qualificação.

Base Legal e Retenção

Tratamos dados com base em consentimento e finalidade de comunicação comercial legítima. Retemos os dados apenas pelo período necessário para obrigações comerciais e legais.

Compartilhamento

Os envios são processados via Formspree e sistemas internos. Não vendemos dados pessoais.

Seus Direitos

Você pode solicitar acesso, correção ou exclusão de dados pelo e-mail contact@azebra.com.

Segurança

Aplicamos medidas técnicas e organizacionais para reduzir risco de acesso não autorizado.
